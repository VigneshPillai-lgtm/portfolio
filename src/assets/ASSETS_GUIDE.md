/* 
 * CUSTOM ASSETS & GRAPHICS GUIDE
 * ================================
 * 
 * This file provides guidance on how to add custom assets to your portfolio
 * to enhance the groovy, modern design similar to Hack the North.
 * 
 * ASSET RECOMMENDATIONS:
 * 
 * 1. HERO SECTION BACKGROUND:
 *    - Add animated SVG shapes or use canvas-based animations
 *    - Consider: geometric patterns, Tech illustrations, glowing orbs
 *    - Recommended: Use Illustration libraries or create custom SVGs
 * 
 * 2. PROJECT IMAGES:
 *    - Add project screenshots/mockups in src/assets/projects/
 *    - Format: WebP or PNG (300x200px minimum)
 *    - Add hover animations to showcase projects
 * 
 * 3. SKILL BADGES/ICONS:
 *    - Create custom skill icons or use icon libraries
 *    - Place in src/assets/icons/
 *    - Animate them with Framer Motion on hover
 * 
 * 4. DECORATIVE ELEMENTS:
 *    - Abstract shapes for section backgrounds
 *    - Floating particles and animated blobs
 *    - SVG patterns for texture
 * 
 * 5. SOCIAL MEDIA ASSETS:
 *    - Add branded social icons
 *    - Create link preview cards
 * 
 * RECOMMENDED TOOLS:
 * - Figma: Design custom graphics
 * - Blender: 3D elements and animations
 * - SVG.js or Spline: Interactive 3D web elements
 * - Storyset: Free customizable illustrations
 * - Unsplash/Pexels: High-quality backgrounds
 * 
 * IMPLEMENTATION TIPS:
 * 1. Optimize all images using tools like TinyPNG or ImageOptim
 * 2. Use responsive images with srcset
 * 3. Lazy load non-critical images
 * 4. Consider using WebP format for better compression
 * 5. Test performance with Lighthouse
 * 
 * ANIMATION IDEAS:
 * - Parallax scrolling effects for hero images
 * - Staggered project card reveals
 * - Glowing borders on hover
 * - Animated progress bars for skill levels
 * - Floating particles in background
 * - Morphing SVG shapes
 */
